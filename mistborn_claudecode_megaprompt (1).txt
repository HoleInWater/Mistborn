### TASK 8 — CREATE THE `check` COMMAND SCRIPT

Create a file at the project root called `check.sh` (and `check.bat` for Windows):

#### check.sh
```bash
#!/bin/bash
# ============================================================
# check.sh — Mistborn Project Health Check
# Run this with: bash check.sh
# ============================================================
# This script scans the codebase for all TODO comments left
# by the AI agent and teammates, prints them grouped by file,
# and shows a summary count.
# ============================================================

echo ""
echo "🌑 MISTBORN ERA ONE — PROJECT CHECK"
echo "======================================"
echo "📅 $(date)"
echo ""

echo "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
echo "📋 TODO COMMENTS IN CODE"
echo "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"

# Find all TODO comments in C# scripts, grouped by file
grep -rn "TODO" Assets/_Project/Scripts/ --include="*.cs" \
  | sed 's|Assets/_Project/Scripts/||' \
  | awk -F: '{
      file=$1;
      if(file != lastfile) {
        print "\n📄 " file;
        lastfile=file;
      }
      print "   Line " $2 ": " $3
    }'

echo ""
echo "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
echo "📊 TODO SUMMARY COUNT"
echo "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
TOTAL=$(grep -rn "TODO" Assets/_Project/Scripts/ --include="*.cs" | wc -l)
AI_TODOS=$(grep -rn "TODO (AI Agent):" Assets/_Project/Scripts/ --include="*.cs" | wc -l)
TEAM_TODOS=$(grep -rn "TODO (Team):" Assets/_Project/Scripts/ --include="*.cs" | wc -l)
echo "  Total TODOs   : $TOTAL"
echo "  AI Agent TODOs: $AI_TODOS  ← things Claude Code still needs to implement"
echo "  Team TODOs    : $TEAM_TODOS  ← decisions your team needs to make"

echo ""
echo "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
echo "🌿 GIT STATUS"
echo "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
git status --short

echo ""
echo "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
echo "🌿 RECENT COMMITS (last 10)"
echo "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
```

**Allomancy Rules (Hard Magic System):**
- Users must INGEST metal physically (swallow metal flakes/dust)
- They then "burn" it — consuming the metal for power
- Power ends when the metal reserve is depleted
- Mistborn can burn all 16 metals
- Mistings can burn exactly ONE metal
- The power comes from Preservation (one of the two Shards of Adonalsium)

**Steel (Steelpushing):**
- Pushes on metals in the Allomancer's line of sight
- Force is applied AWAY from the Allomancer
- Newton's 3rd Law: if target is heavier/anchored, the Allomancer moves instead
- Used for: launching coins as projectiles, pushing metal enemies,
  pushing off metal surfaces to fly/leap
- Iconic use: standing on a metal surface and Steelpushing downward = flight

**Iron (Ironpulling):**
- Pulls on metals TOWARD the Allomancer
- Same Newton's 3rd Law applies: heavy anchor = Allomancer pulled toward it
- Used for: disarming enemies, pulling self toward metal anchors to swing/climb
- Complementary to Steel (same "line" of metals — physical external pushing metals)

**Allomantic Sight:**
- When burning Steel or Iron, the Allomancer sees faint blue lines
- Lines extend from their chest to every metal object in range
- Line thickness/brightness indicates mass of the object
- Critical for targeting during push/pull

**Metal Reserves:**
- Burning faster = more power but depletes faster
- Flaring: burning at max power, faster depletion, stronger effect
- A Mistborn manages 16 separate reserves simultaneously
