# Character Progression Design

## Overview

How the player grows and unlocks abilities over the course of the game.

---

## Progression Systems

### 1. Allomancy Mastery
Learn to burn more metals, burn them better.

**Unlocks:**
- Start with: Steel, Iron
- Unlock at story beats: Pewter, Tin, Zinc, Brass
- Later unlocks: Copper, Bronze, Gold, Electrum
- Late game: Duralumin, Aluminum, others

**Mastery Levels per Metal:**
| Level | Name | Bonus |
|-------|------|-------|
| 1 | Basic | Can burn, slow drain |
| 2 | Trained | Normal drain |
| 3 | Skilled | Efficient (slow drain) |
| 4 | Expert | Can flare |
| 5 | Master | Reduced drag, max efficiency |

**How to Gain:**
- Use the metal in combat
- Complete metal-specific challenges
- Story progression

---

### 2. Traversal Skills
Learn to move better through the world.

**Skills:**
| Skill | Effect | How to Unlock |
|-------|--------|--------------|
| Double Jump | One extra jump | Find hidden metal, use it |
| Wall Push | Push off walls | Push 50 metal objects |
| Air Push | Push mid-air | Chain 5 pushes in one flight |
| Precision Pull | Accurate pull target | Pull 100 coins to you |
| Long Push | Extended range | Push 20 objects at max distance |

---

### 3. Combat Techniques
Better ways to hurt people.

**Techniques:**
| Technique | Effect | Requirement |
|-----------|--------|-------------|
| Coin Spray | Burst of 5 coins | Throw 50 coins |
| Precision Push | Extra knockback | Push 30 disarmed weapons |
| Chain Pull | Pull 3 things at once | Pull 100 metal objects |
| Bubble Deflect | Block projectiles | Use steel while being shot |
| Momentum Build | Charge damage with speed | Fly into 10 enemies |

---

### 4. Metal Reserves
Carry more, burn longer.

**Upgrades:**
- **Reserve Size:** More units of each metal
- **Recovery Rate:** Regenerate small amounts over time
- **Efficiency:** Learn to burn less while maintaining effect

**Base Values:**
| Metal | Starting Reserve | Starting Drain |
|-------|------------------|----------------|
| Steel | 100 | 2/sec |
| Iron | 100 | 2/sec |
| Pewter | 50 | 3/sec |
| Tin | 50 | 2/sec |

---

## XP and Leveling

### XP Sources
- Defeating enemies
- Finding metal caches
- Completing challenges
- Story progression

### Level System
| Level | Total XP | Unlocks |
|-------|----------|---------|
| 1 | 0 | Steel, Iron |
| 2 | 100 | Pewter |
| 3 | 300 | Tin |
| 4 | 600 | Copper |
| 5 | 1000 | Zinc, Brass |
| 6 | 1500 | Bronze |
| 7 | 2200 | Gold |
| 8 | 3000 | Electrum |
| 9 | 4000 | Duralumin |
| 10 | 5500 | Full Mistborn |

## Skill Trees

### Combat Tree
```
                    [Pewter Master]
                         ↑
[Pewter Basics] → [Pewter Efficiency] → [Pewter Rage]
      ↓
[Tin Basics] → [Tin Resistance] → [Tin Immunity]
```

### Traversal Tree
```
[Wall Push] → [Air Control] → [Momentum Flight]
      ↓
[Precision Landing] → [Silent Flight] → [Shadow Movement]
```

### Allomancer Tree
```
[Reserve Boost] → [Efficient Burning] → [Metal Sense]
      ↓
[Flaring] → [Precision Flaring] → [Duralumin Burst]
```

## Character Builds

### The Coinshot
Focus: Steel mastery, ranged damage
- Max Steel efficiency
- Coin Spray technique
- Precision Push

### The Lurcher
Focus: Iron mastery, crowd control
- Max Iron range
- Chain Pull
- Environmental kills

### The Tank
Focus: Pewter, survivability
- Pewter efficiency
- High HP upgrades
- Pain immunity

### The Scout
Focus: Traversal, stealth
- All traversal skills
- Tin resistance
- Silent movement

### Full Mistborn (End Game)
All metals, all techniques, maximum versatility.
